How to use these icons

This gallery contains downloadable PNG and SVG icons that can be easily used in PowerPoint, Word, Canva, and other applications.

  1. Browse the gallery and use the filters or search bar to find the icon you want.
  2. Click the PNG or SVG download button under any icon.
  3. Once downloaded, open your PowerPoint or Word document and go to Insert → Pictures → This Device to add the image.
  4. You can resize, move, or recolor the icon in your document as needed.
Tip: For sharp, scalable graphics, use SVG if your software supports it (PowerPoint 2016+ on Windows or Microsoft 365). Use PNG for older versions or fixed-size images.
